◆ addNewCourse()

void StudentDb::addNewCourse ( )

Method that adds a new course into the mapped entity of 'courses'.

The user is asked to provide : course type (Block/Weekly), course key, course title, major and credit points as basic information.

if course is a block course, then the user is also asked for start date, end date, start time and end time of the course.

if course is a weekly course, then the user is also asked for day of the course in the week, start time and end time of the course.

The user is asked to provide the details of students namely, first and last name, date of birth as day, month and year, address information like street address name, postal code and city name and any additional information for address.

void StudentDb::printStudent ( )

Method that prints a student's complete information.

The user is asked to enter the matrikel number of the student whose information needs to be printed.

Basic information like personal and address information are printed by default.

Enrollment information of any of the courses is printed only if the student has been enrolled.

void StudentDb::searchStudent ( )

Method that searches a student in the student database.

The user is asked to provide atleast a part of the string associated with the first or last name or both of the student.
